# Setup
Install Node.js : (https://nodejs.org/en/download)

After download complete open terminal and start run 
```
npm install
```

and real-time update 
```
npm install -g nodemon 
```
Install Express.js and ejs 
```
npm install express
npm install express-session
npm install ejs
```
